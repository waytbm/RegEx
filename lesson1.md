# Регулярные выражения (Regular Expressions)

## Введение в регулярные выражения на Python

Регулярные выражения (Regular Expressions, сокращенно RegEx или RegExp) — это специальный язык для описания шаблонов поиска в тексте.

### Вот 3 самые частые задачи, которые вы будете решать:

1. Валидация данных (Проверка)
Пользователь ввел email при регистрации. Нам нужно проверить, действительно ли это почта, а не просто набор букв.

*/ Без RegEx: Написать 20 строк кода с кучей условий if и циклов.

*/ С RegEx: Одна строка кода с шаблоном ^\S+@\S+\.\S+$.

2. Поиск и извлечение (Парсинг)
У вас есть лог-файл сервера с миллионом строк. Нужно вытащить только IP-адреса тех, кто пытался взломать систему. RegEx позволяет "выудить" из текста нужные кусочки данных.

3. Замена и очистка
Удалить все HTML-теги из текста, оставив только слова? Заменить формат даты с 2023-10-05 на 05.10.2023 во всем документе разом? Для RegEx это дело пары секунд.

**Модуль re**
Python имеет встроенную поддержку регулярных выражений через стандартный модуль re. 


Пример регулярного выражения 

```python
import re

text = "Карта map и объект bitmap - это разные вещи"

match = re.findall("map", text)
print(match)
```


Результат:

```python
['map', 'map']
```

В данном примере разобрана функция findall, принимающая два аргумента: что искать, где искать.
